Handy 개인정보처리방침

Handy는 사용자가 직접 입력한 숭실대학교 LMS 계정 정보와, 로그인 이후 표시되는 학습 정보를 어떻게 처리하는지 안내합니다.

개발자 서버 저장 없음
기기 내 저장 위치 Android Keystore / iOS Keychain
추적 및 광고 SDK 사용하지 않음

1. 수집 및 처리 정보

앱이 처리하는 정보

  • 필수 입력 정보: 숭실대학교 LMS 아이디, 비밀번호
  • 자동 조회 정보: 과목명, 출석 현황, 공지, 할 일, 과제 점수 등 LMS 화면 정보

입력된 로그인 정보는 사용자가 요청한 로그인 인증과 자동 로그인 기능 제공을 위해 처리됩니다.

2. 처리 목적

정보를 사용하는 이유

  • 숭실대학교 LMS 로그인 인증
  • 기기 내 저장을 통한 자동 로그인 제공
  • LMS에서 조회한 학습 정보를 앱 화면에 표시

3. 보관 방식

저장 및 보관

Handy는 사용자가 입력한 LMS 로그인 정보를 개발자 자체 서버에 저장하지 않습니다. 로그인 정보는 사용자 기기 내부의 보안 저장소에만 보관됩니다.

  • Android: Keystore 기반 암호화 후 로컬 저장소에 보관
  • iOS: Keychain에 로그인 정보 저장

로그인 이후 조회되는 학습 정보는 앱 사용 중 화면 표시를 위해 처리되며, 현재 버전에서는 별도의 장기 저장소나 분석 시스템에 축적하지 않습니다.

4. 외부 전송 및 제3자 제공

통신 범위

로그인 시 입력한 아이디와 비밀번호는 사용자가 요청한 인증을 수행하기 위해 숭실대학교 LMS 서비스와 통신하는 데 사용됩니다.

Handy는 광고, 마케팅, 제3자 행동 추적을 목적으로 개인정보를 판매하거나 제공하지 않습니다.

5. 추적 및 광고

추적 도구 사용 여부

Handy는 광고 식별자 수집, 분석 추적 SDK, 사용자 행동 프로파일링 도구를 사용하지 않습니다.

6. 이용자 선택권

삭제 및 철회 방법

  • 앱 내 저장 정보 삭제: 로그인 화면의 저장 정보 삭제 또는 로그아웃 기능 사용
  • 앱 삭제: 기기 내 저장된 앱 데이터가 함께 제거될 수 있음
  • LMS 계정 자체의 생성, 정지, 삭제는 숭실대학교 LMS 및 학교 계정 정책에 따름
