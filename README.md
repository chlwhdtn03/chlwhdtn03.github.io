### Store Metadata Pages

Static web pages for store submission are available under [`/docs`](./docs):

- [`/docs/privacy/index.html`](./docs/privacy/index.html): privacy policy
- [`/docs/support/index.html`](./docs/support/index.html): support information
- [`/docs/index.html`](./docs/index.html): landing page linking both documents

To publish these pages with GitHub Pages:

1. Push this repository to GitHub.
2. In the repository settings, open `Pages`.
3. Set the source to your deployment branch and choose the `/docs` folder.
4. After GitHub Pages is enabled, use the generated public URLs for the App Store Connect `Privacy Policy URL`, `Support URL`, and the Google Play privacy policy link.
